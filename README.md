# Patch The Blockheads server Linux binary for Ubuntu 26.04

This repository contains scripts for patching The Blockheads server Linux binary for systems running Ubuntu 26.04. Unofficial and unsupported by the developers of The Blockheads.

```sh
git clone https://github.com/JarlPenguin/patch-blockheads-server
cd patch-blockheads-server
sudo ./install-deps.sh
./patch-binary.sh
```
